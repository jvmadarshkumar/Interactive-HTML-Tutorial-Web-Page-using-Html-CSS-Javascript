Core Structure of HTML

At its core, an HTML document is structured in a way that browsers can read and render it correctly. The basic structure looks something like this:

            <!DOCTYPE html>
            <html>
            <head>
                <title>Page Title</title>
            </head>
            <body>
                <!-- Content goes here -->
            </body>
            </html>

What are Tags?

Tags are the building blocks of HTML. They are used to create elements such as headings, paragraphs, links, and more. Tags usually come in pairs: an opening tag and a closing tag. The content between the tags is the element content.

Example:

            <p>This is a paragraph.</p>

Here, <p> is the opening tag, and </p> is the closing tag. The content "This is a paragraph." is the element content.

General Syntax

HTML tags are enclosed in angle brackets (< and >). Most tags come in pairs with a start tag and an end tag, but some tags, like the <img> tag, are self-closing.

Including Meta Tags

Meta tags are snippets of text that describe a page's content; they don't appear on the page itself but only in the page's code. Meta tags are used to provide metadata about the HTML document.

Example with Meta Tags:

            <!DOCTYPE html>
            <html>
            <head>
                <meta charset="UTF-8">
                <meta name="viewport" content="width=device-width, initial-scale=1.0">
                <meta name="description" content="This is an example of a meta description tag.">
                <meta name="keywords" content="HTML, CSS, JavaScript">
                <title>Page Title</title>
            </head>
            <body>
                <h1>Welcome to My Website</h1>
                <p>This is a simple HTML document.</p>
            </body>
            </html>
