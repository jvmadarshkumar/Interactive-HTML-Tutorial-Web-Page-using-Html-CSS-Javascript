HTML Head Tag Tutorial

Basic Structure

The <head> tag in HTML contains metadata about the document. Hereâ€™s a basic structure:

        <!DOCTYPE html>
        <html lang="en">
        <head>
            <meta charset="UTF-8">
            <title>Document Title</title>
        </head>
        <body>
            <!-- Page content goes here -->
        </body>
        </html>

Common Elements in <head>

<title>

Sets the title of the document, shown in the browser's title bar or tab.

        <title>My Website</title>

<meta>

Provides metadata such as character set, author of the document, and keywords for search engines.

        <meta charset="UTF-8">
        <meta name="description" content="A brief description of the page">
        <meta name="keywords" content="HTML, CSS, JavaScript">
        <meta name="author" content="Your Name">
        <meta name="viewport" content="width=device-width, initial-scale=1.0">

<link>

Used to link to external resources like CSS files.

        <link rel="stylesheet" href="styles.css">

<style>

Contains internal CSS styles.

        <style>
            body {
                font-family: Arial, sans-serif;
                background-color: #f4f4f4;
            }
        </style>

<script>

Includes JavaScript code or links to an external JavaScript file.

        <script src="script.js"></script>

<base>

Specifies the base URL for all relative URLs in the document.

        <base href="https://www.example.com/">

Complete Example

        <!DOCTYPE html>
        <html lang="en">
        <head>
            <meta charset="UTF-8">
            <meta name="viewport" content="width=device-width, initial-scale=1.0">

            <meta name="author" content="Your Name">
            <title>My Website</title>
            <link rel="stylesheet" href="styles.css">
            <style>
                body {
                    font-family: Arial, sans-serif;
                    background-color: #f4f4f4;
                }
            </style>
            <script src="script.js"></script>
        </head>
        <body>
            <h1>Welcome to My Website</h1>
            <p>This is an example of a web page with a head section that includes metadata, styles, and scripts.</p>
        </body>
        </html>
