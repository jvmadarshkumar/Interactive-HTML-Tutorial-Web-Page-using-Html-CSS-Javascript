Links and Images with Maps in HTML

Links in HTML

Links are created using the <a> tag. The href attribute specifies the URL of the page the link goes to.

<a href="https://www.example.com">Visit Example.com</a>
Visit Example.com

Images in HTML

Images are embedded in HTML using the <img> tag. The src attribute specifies the path to the image, and the alt attribute provides alternative text for the image.

<img src="path/to/image.jpg" alt="Description of the image">

Image Maps in HTML

Image maps allow you to create clickable areas on an image. Use the <map> element along with the <area> elements to define the clickable areas.

<img src="path/to/image-map.jpg" alt="Image Map" usemap="#imagemap">

<map name="imagemap">
    <area shape="rect" coords="34,44,270,350" alt="Clickable Area 1" href="https://www.example1.com">
    <area shape="circle" coords="500,300,100" alt="Clickable Area 2" href="https://www.example2.com">
    <area shape="poly" coords="120,34,215,50,90,100" alt="Clickable Area 3" href="https://www.example3.com">
</map>

Image Map Example

This example demonstrates how to use different shapes for clickable areas on an image map.

Explanation of Attributes

shape: Defines the shape of the clickable area. Can be rect, circle, or poly.

coords: Specifies the coordinates for the shape.

  • For a rectangle, use left-x,top-y,right-x,bottom-y.
  • For a circle, use center-x,center-y,radius.
  • For a polygon, use a series of x,y coordinates.

href: The URL to link to when the area is clicked.

alt: Alternative text for the area.
