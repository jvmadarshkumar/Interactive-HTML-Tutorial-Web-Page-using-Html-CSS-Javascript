HTML Formatting Styles and Global Attributes

HTML Formatting Styles

Formatting elements are used to change the appearance of the text. Here are some common ones:

This text is bold.

<b>This text is bold.</b>

This text is strong.

<strong>This text is strong.</strong>

This text is italic.

<i>This text is italic.</i>

This text is emphasized.

<em>This text is emphasized.</em>

This text is underlined.

<u>This text is underlined.</u>

This text is struck through.

<s>This text is struck through.</s>

This text is deleted.

<del>This text is deleted.</del>

H2O

H<sub>2</sub>O

x2

x<sup>2</sup>

This is inline code.

<code>This is inline code.</code>
        <pre>
        This is preformatted text.
            It preserves spaces and line breaks.
        </pre>

HTML Global Attributes

Global attributes can be used on any HTML element. Here are some common ones:

This is a paragraph with a class.

<p class="intro">This is a paragraph with a class.</p>

This is a paragraph with an id.

<p id="unique">This is a paragraph with an id.</p>

This is a blue paragraph.

<p style="color: blue;">This is a blue paragraph.</p>

Hover over this paragraph to see the tooltip.

<p title="I am a tooltip">Hover over this paragraph to see the tooltip.</p>

This is an English paragraph.

<p lang="en">This is an English paragraph.</p>

This paragraph has custom data.

<p data-user="john_doe">This paragraph has custom data.</p>
