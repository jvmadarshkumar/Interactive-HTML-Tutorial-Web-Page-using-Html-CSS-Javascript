HTML Forms and Attributes

Basic Form Structure

HTML forms are used to collect user input. A form can contain various input elements like text fields, checkboxes, radio buttons, submit buttons, and more.

<form action="/submit-form" method="post">
    <label for="name">Name:</label>
    <input type="text" id="name" name="name">
    
    <label for="email">Email:</label>
    <input type="email" id="email" name="email">
    
    <input type="submit" value="Submit">
</form>

Form Attributes

action

Specifies where to send the form data when the form is submitted.

<form action="/submit-form" method="post">

method

Specifies the HTTP method to use when sending form data. Can be get or post.

<form action="/submit-form" method="post">

target

Specifies where to display the response after submitting the form.

<form action="/submit-form" method="post" target="_blank">

enctype

Specifies how the form data should be encoded when submitting it to the server (for file uploads).

<form action="/submit-form" method="post" enctype="multipart/form-data">
