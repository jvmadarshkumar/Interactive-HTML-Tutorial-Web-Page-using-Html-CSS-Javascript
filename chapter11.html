HTML Semantics Example

Header

The <header> element represents a group of introductory or navigational aids.

        <header>
            <h1>Welcome to My Website</h1>
            <nav>
                <ul>
                    <li><a href="#home">Home</a></li>
                    <li><a href="#about">About</a></li>
                    <li><a href="#contact">Contact</a></li>
                </ul>
            </nav>
        </header>

Navigation

The <nav> element represents a section of a page intended for navigation.

        <nav>
            <ul>
                <li><a href="#home">Home</a></li>
                <li><a href="#about">About</a></li>
                <li><a href="#services">Services</a></li>
                <li><a href="#contact">Contact</a></li>
            </ul>
        </nav>

Main

The <main> element represents the dominant content of the <body>.

        <main>
            <h2>Main Content</h2>
            <p>This is the main section of the webpage.</p>
        </main>

Section

The <section> element represents a standalone section that doesn’t have a more specific semantic element.

        <section>
            <h2>Section Title</h2>
            <p>This is a section within the main content.</p>
        </section>

Article

The <article> element represents a self-contained composition in a document.

        <article>
            <h2>Article Title</h2>
            <p>This is an article.</p>
        </article>

Aside

The <aside> element represents content tangentially related to the content around it.

        <aside>
            <h2>Related</h2>
            <p>This is some related information.</p>
        </aside>

Footer

The <footer> element represents the footer of a section or document.

        <footer>
            <p>© 2024 My Website</p>
        </footer>

Figure and Figcaption

The <figure> and <figcaption> elements represent self-contained content with an optional caption.

        <figure>
            <img src="image.jpg" alt="Description of the image">
            <figcaption>Image Caption</figcaption>
        </figure>
