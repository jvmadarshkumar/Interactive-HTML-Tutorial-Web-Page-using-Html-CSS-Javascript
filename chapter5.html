Value of Colors and CSS Formats in HTML

Value of Colors in HTML

HTML allows you to specify colors in several ways:

Name: There are 140 color names predefined in HTML.

<p style="color: red;">This text is red.</p>

Example: This text is red.

Hexadecimal: A six-digit code representing the intensity of red, green, and blue.

<p style="color: #ff0000;">This text is red.</p>

Example: This text is red.

RGB: Defines color using the red, green, and blue components.

<p style="color: rgb(255, 0, 0);">This text is red.</p>

Example: This text is red.

RGBA: Like RGB but includes an alpha channel (transparency).

<p style="color: rgba(255, 0, 0, 0.5);">This text is semi-transparent red.</p>

Example: This text is semi-transparent red.

HSL: Defines colors using hue, saturation, and lightness.

<p style="color: hsl(0, 100%, 50%);">This text is red.</p>

Example: This text is red.

HSLA: Like HSL but includes an alpha channel.

<p style="color: hsla(0, 100%, 50%, 0.5);">This text is semi-transparent red.</p>

Example: This text is semi-transparent red.

CSS Format in HTML

Inline CSS: Directly in the HTML elements.

Example: This is a blue paragraph with a font size of 20px.

Internal CSS: Within the <style> tag in the <head> section of the HTML document.

<style>
    p {
        color: blue;

    }
</style>

Example: This is a blue paragraph with a font size of 20px.

External CSS: In an external CSS file. The file is linked using the <link> tag in the <head> section.

<link rel="stylesheet" href="styles.css">

    color: blue;
