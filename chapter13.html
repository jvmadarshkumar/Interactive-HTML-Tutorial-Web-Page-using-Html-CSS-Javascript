HTML Input Types

Text Input

Allows users to enter text.

        <label for="text">Text Input:</label>
        <input type="text" id="text" name="text">

Password Input

Hides the text input, often used for passwords.

        <label for="password">Password:</label>
        <input type="password" id="password" name="password">

Email Input

Specifically for email addresses.

        <label for="email">Email:</label>
        <input type="email" id="email" name="email">

Number Input

For numerical input.

        <label for="number">Number:</label>
        <input type="number" id="number" name="number">

Date Input

Allows users to select a date.

        <label for="date">Date:</label>
        <input type="date" id="date" name="date";

Time Input

Allows users to select a time.

        <label for="time">time Input:</label>
        <input type="time" id="time" name="time">

File Input

For file upload.

        <label for="file">Upload File:</label>
        <input type="file" id="password" name="file">

Checkbox

Allows users to select multiple options .

        <label for="subscribe">Subscribe to newsletter:</label>
        <input type="checkbox" id="checkbox" name="checkbox">

Radio Button

Allows user to select one option from a set.

        <label for="male">Male</label>
        <input type="radio" id="male" name="gender" value="male">
        <label for="female">Female</label>
        <input type="radio" id="female" name="gender" value="female">

Range Slider

Lets user select a value from a range.

        <label for="range">Range:</label>
        <input type="range" id="range" name="range" min="0" max="100">

Submit Button

To submit the form.

        <input type="submit" value="submit">
