HTML Table Tutorial

Basic Table Structure

                    
        <table>
          <tr>
            <th>Header 1</th>
            <th>Header 2</th>
          </tr>
          <tr>
            <td>Row 1, Cell 1</td>
            <td>Row 1, Cell 2</td>
          </tr>
          <tr>
            <td>Row 2, Cell 1</td>
            <td>Row 2, Cell 2</td>
          </tr>
        </table>

Key Attributes

<table>: This tag defines the table.
Attributes:

  • border: Specifies the width of the border around table cells. Example: <table border="1">
  • cellpadding: Specifies the space between the cell content and the cell borders. Example: <table cellpadding="10">
  • cellspacing: Specifies the space between cells. Example: <table cellspacing="10">
  • width: Specifies the width of the table. Example: <table width="100%">
  • height: Specifies the height of the table. Example: <table height="200px">

<tr>: This tag defines a row in the table.
<th>: This tag defines a header cell in the table. It is usually bold and centered by default.
Attributes:

  • colspan: Specifies the number of columns a header cell should span. Example: <th colspan="2">
  • rowspan: Specifies the number of rows a header cell should span. Example: <th rowspan="2">

<td>: This tag defines a standard cell in the table.
Attributes:

  • colspan: Specifies the number of columns a cell should span. Example: <td colspan="2">
  • rowspan: Specifies the number of rows a cell should span. Example: <td rowspan="2">
  • align: Specifies the horizontal alignment of the cell content. Values can be left, center, or right. Example: <td align="center">
  • valign: Specifies the vertical alignment of the cell content. Values can be top, middle, bottom, or baseline. Example: <td valign="top">
