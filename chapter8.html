HTML Elements

Lists in HTML

HTML supports three types of lists:

Ordered List (<ol>):
This list is numbered. 
<ol>
    <li>First item</li>
    <li>Second item</li>
    <li>Third item</li>
</ol>
  1. First item
  2. Second item
  3. Third item


Unordered List (<ul>):
This list uses bullets. 
<ul>
    <li>First item</li>
    <li>Second item</li>
    <li>Third item</li>
</ul>
  • First item
  • Second item
  • Third item


Unordered List (<dl>):
This list is used for terms and descriptions. 
<dl>
    <dt>Term 1</dt>
    <dd>Description for Term 1</dd>
    <dt>Term 2</dt>
    <dd>Description for Term 2</dd>
</dl>
Term 1
Description for Term 1
Term 2
Description for Term 2



Inline Elements

Inline elements do not start on a new line and only take up as much width as necessary

<a>:
Defines a hyperlink 
<a href="https://www.example.com">Visit Example</a>
<span>:
Used to group inline-elements 
<span style="color: red;">This text is red.</span>
<img>:
Embeds an image 
<img src="image.jpg" alt="An image">
Visit Example
This text is red.



Block Elements

Block elements start on a new line and take up the full width available.

<div>:
Used to group block-level content 
<div style="background-color: lightgray;">This is a div element.</div>
<p>:
Defines a paragraph 
<p>This is a paragraph.</p>
<h1> to <h6>:
Defines HTML headings 
<h1>This is a Heading 1</h1>
<h2>This is a Heading 2</h2>
This is a div element.

This is a paragraph.

This is a Heading 1

This is a Heading 2
