HTML Basics

What are Tags?

Tags are the building blocks of HTML. They define elements that structure your web page. Tags typically come in pairs: an opening tag and a closing tag.

Example:

<p>This is a paragraph.</p>

What are Elements?

Elements are the combination of the opening tag, content, and closing tag. They represent the content in your web page.

Example:

<p>This is a paragraph.</p>

What are Attributes?

Attributes provide additional information about an element. They are always included in the opening tag and usually come in name/value pairs like name="value".

Example:

<a href="https://www.example.com">Visit Example.com</a>

What is a Tag Name?

The tag name is the word inside the angle brackets. It specifies the type of element.

Example:

<h1>This is a heading</h1>

Paragraph Tag

<p>This is a paragraph. It is used to group sentences and sections of text together. Paragraphs are block-level elements, which means they start on a new line and take up the full width available.</p>
