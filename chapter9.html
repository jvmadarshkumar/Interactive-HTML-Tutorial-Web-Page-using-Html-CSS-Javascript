HTML Class ID, and iframe

Class

The class attribute is used to specify one or more class names for an HTML element. Class names can be used to target elements with CSS or JavaScript.

        <p class="intro">This is a paragraph with a class.</p>

This is a paragraph with a class.

The class attribute can also contain multiple classes:

        <p class="intro highlight">This paragraph has multiple classes.</p>

This paragraph has multiple classes.

ID

The id attribute specifies a unique id for an HTML element. The id must be unique within the HTML document and can be used to target the element with CSS or JavaScript.

        <p id="unique">This is a paragraph with an id.</p>

This is a paragraph with an id.

iframe

The <iframe> element is used to embed another HTML page within the current page. It acts as an inline frame, allowing you to include content from another source.

        <iframe src="https://www.example.com" width="600" height="400" title="Example iframe"></iframe>

The following is an iframe embedding another webpage:

iframe Attributes

The <iframe> element has several attributes that you can use to customize its behavior:

  • src: Specifies the URL of the page to embed.
  • width: Specifies the width of the iframe.
  • height: Specifies the height of the iframe.
  • title: Provides an accessible name for the iframe.
